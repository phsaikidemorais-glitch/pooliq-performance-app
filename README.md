# PoolIQ - AI Swimming Performance Platform

PoolIQ is a Flask-based web application designed to help swimmers and coaches analyze training performance, readiness, pace, workload, and shoulder risk.

## Features

- User registration and login
- Training session tracking
- Swim set logging
- Readiness score calculation
- Pace and workload analysis
- Shoulder risk detection
- Smart coaching insights
- Training history dashboard

## Tech Stack

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript

## Project Vision

PoolIQ is built to become a premium swimming performance platform that connects training data, athlete readiness, and coaching intelligence.

The goal is to help swimmers and coaches make smarter decisions based on real training context.
